Aqui estão os conceitos básicos sobre como o DOM funciona:

O DOM (Document Object Model) é uma representação hierárquica e estruturada de um documento HTML ou XML, que pode ser manipulada por linguagens de script, como JavaScript. O DOM representa a estrutura do documento como uma árvore de objetos, onde cada nó na árvore representa um elemento, atributo, texto ou qualquer outra parte do documento.

  • 1
    • Árvore de Documento: O DOM organiza o documento em uma árvore de objetos. O nó superior é chamado de "nó de documento" e representa todo o documento. Cada elemento, atributo, texto, etc., no documento é representado por um nó na árvore.
    • Elementos e Atributos: Elementos HTML são representados por nós na árvore. Atributos dos elementos são representados como propriedades dos nós.
    • Manipulação com JavaScript: JavaScript é frequentemente usado para interagir com o DOM e fazer alterações dinâmicas na página. Métodos como getElementById, getElementsByClassName, querySelector, etc., são usados para obter referências a elementos específicos no DOM.
    • Nós, Pais e Filhos: Cada nó no DOM pode ter "filhos" e "pais", exceto o nó de documento que é o nó raiz. Os métodos como parentNode, childNodes, firstChild, lastChild, etc., são usados para navegar pela árvore.
    • Atualização Dinâmica: JavaScript pode ser usado para modificar o conteúdo, estrutura ou estilo do documento em tempo de execução. Por exemplo, você pode adicionar, remover ou modificar elementos no DOM.

      • Neste exemplo, o JavaScript obtém uma referência ao elemento com o ID "titulo" e, em seguida, modifica o conteúdo desse elemento usando a propriedade innerHTML. Isso é apenas um exemplo simples; a manipulação do DOM pode ser muito mais complexa, dependendo dos requisitos da aplicação.
